# Nelly

A simple student and teacher portal with a clean login page, timetable display, course list, and PDF download links.

## What is included

- Student and teacher login options
- Timetable view for each role
- List of courses offered
- Downloadable PDF resources
- Modern page design with smooth animations

## Files

- `index.html` — portal front end
- `index.css` — page styling and animations
- `index.php` — optional PHP version with server-side login handling

## Login details

- Student: `student1 / User2024`
- Teacher: `teacher1 / User2024`

## How to use

Open `index.html` in a browser to try the front-end version.

For the PHP version, run a local PHP server and open `index.php`.
